## About

Image to Data URI is a tool to convert images to Data URI. Data URI is a URI scheme that provides a way to include data in-line in web pages as if they were external resources.

It is a very useful tool for web developers. It is used to embed images in HTML and CSS files.
It is also used to embed small files like JSON, XML, etc. in HTML and CSS files. It is also used to embed small files like JSON, XML, etc. in HTML and CSS files.

## How to use

1. Upload your Image file by clicking the "Choose File" button.
2. Click "Copy" button to copy the Data URI to your clipboard.
