## About

SVG To PNG is a free online tool to convert SVG Images to PNG Images. No limits, no watermarks, free to use for everyone.

## How to use

1. Upload your SVG file by clicking the "Choose File" button.
2. Click the "Convert" button to start the conversion.
3. When the conversion is done, click the "Save" button to save the converted file.
