## About

SVG to JPG/JPEG is a freeonline tool that converts SVG files to JPG/JPEG images. The tool is realy easy to use. Our clean and minimalistic ui design makes it realy easy to understand and use the tool. We do not limit the size of file you upload. Upload and convert unlimited files with unlimited size write into your browser. All the conversion happens write into your browser. No image/images is sent to the server. So your data is secure.

## How to use

1. Upload your SVG file by clicking the "Choose File" button.
2. Click the "Convert" button to start the conversion.
3. When the conversion is done, click the "Save" button to save the converted JPG/JPEG file.
