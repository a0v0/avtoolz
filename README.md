# aVToolz v0.0.0 (Deprecated)

## Requirements

- [Node.js](https://nodejs.org/en/)
- [pnpm](https://pnpm.io/)

## Running Locally

1. Clone the repository

```bash
git clone https://github.com/a0v0/avtoolz --branch v0.0.1 --single-branch
```

2. Install dependencies

```bash
pnpm install
```

3. Run the development server

```bash
pnpm start
```
