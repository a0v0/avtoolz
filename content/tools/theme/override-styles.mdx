# Override styles

NextUI allows you to override the default styles of the components in several ways. Before you start overriding
the NextUI components styles we recommend you to read the [Stitches Styling docs](https://stitches.dev/tools/styling) and
[Stitches variants docs](https://stitches.dev/tools/variants).

Remember that all [Stitches](https://stitches.dev) features are available in NextUI since NextUI uses Stitches as a style engine.

<CarbonAd />

<Spacer y={2} />

Here are some options depending on your goals:

- **Using the css prop**
- **Using the styled function**

> **Note:** The tokens used here like `$space$1`, `$pink800`, etc.. are defined in the [default theme](/tools/theme/default-theme) section.

<Spacer y={2} />

### Using the css prop

NextUI provides the Stitches `css` prop for overriding styles easily.
It's like the style attribute, but it supports tokens, media queries, nesting and token-aware values.

```jsx
import { css, Button } from "@nextui-org/react";
```

<Playground
  initialEditorOpen
  code={`
<Button 
  auto 
  css={{ 
    borderRadius: '$xs', // radii.xs
    border: '$space$1 solid transparent',
    background: '$pink800', // colors.pink800
    color: '$pink100',
    height: '$12', // space[12]
    boxShadow: '$md', // shadows.md
    '&:hover': {
      background: '$pink100',
      color: '$pink800',
    },
    '&:active': {
      background: '$pink200',
    },
    '&:focus': {
      borderColor: '$pink400',
    },
  }}
  >
    Custom button
</Button>
`}
/>

> For more information about the Stitches css prop, check out the [Stitches documentation](https://stitches.dev/tools/overriding-styles#the-css-prop).

### Using the styled function

NextUI provides the Stitches `styled` function for creating your own components easily.
The `styled` function allows you to create a component including its styles and variants. It receives:

- `element` or `component`: a HTML element (`div`) or a React Component (`Component`)
- `styles`: as many styleObject as you need

```jsx
import { styled, Button } from "@nextui-org/react";
```

<Playground
  initialEditorOpen
  code={`() => {
const MyStyledButton = styled(Button, {
  boxShadow: '$md', // shadows.md
  variants: {
    size: {
      mysize: {
        height: '$12', // space[12]
        borderRadius: '$xs' // radii.xs
      }
    },
    color: {
      mycolor: {
        background: '$green800', // colors.green800
        color: '$green100', 
        border: '$space$1 solid transparent',
        '&:hover': {
          background: '$green100',
          color: '$green800',
        },
        '&:active': {
          background: '$green200',
        },
        '&:focus': {
          borderColor: '$green400',
        },
      }
    }
  }
});
return (
    <MyStyledButton 
      auto 
      size="mysize"
      color="mycolor">
        Custom button
    </MyStyledButton>
  ) 
}`}
/>

> **Note:** This approach requires you to override and create your own variants like `size` and `color`.

> For more information about the Stitches css prop, check out the [Stitches documentation](https://stitches.dev/tools/api#styled).

### Overriding the HTML tag

NextUI provides an `as` prop for changing which tag a component outputs.
