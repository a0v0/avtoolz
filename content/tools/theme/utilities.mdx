# Utilities

NextUI provides a set of amazing out of the box [Stitches](https://stitches.dev/tools/utils) utilities. Once
you get used to them, you can save a lot of time writing css-in-js code.

<CarbonAd />

<Spacer y={2} />

[Stitches](https://stitches.dev/tools/utils) allows you to create a set of utilities that can be used for styling components so
we create a custom set that can be used for styling the NextUI components.

> Unfortuantely, NextUI doesn't support custom utilities yet, we hope to add this feature in the future.

### Usage

You can use any of the utilities through the `css` prop or using the `styled` function, these properties are detailed [here](/tools/theme/override-styles)

### Theme Map

NextUI tokens types are automatically mapped to CSS Properties for an improved developer experience.

Here the detailed list of all the mapped tokens:

```js
export const themeMap = {
  ...defaultStitchesThemeMap,
  width: "space",
  height: "space",
  minWidth: "space",
  maxWidth: "space",
  minHeight: "space",
  maxHeight: "space",
  flexBasis: "space",
  gridTemplateColumns: "space",
  gridTemplateRows: "space",
  blockSize: "space",
  minBlockSize: "space",
  maxBlockSize: "space",
  inlineSize: "space",
  minInlineSize: "space",
  maxInlineSize: "space",
  borderWidth: "borderWeights",
};
```

> You can read [this](https://stitches.dev/tools/tokens#property-mapping) for more information about `defaultStitchesThemeMap`.
